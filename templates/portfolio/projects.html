Personal

Edge cases like partial fills and identical price orders are all handled properly.

Holiday Cluedo PWA

Personal

Built a Progressive Web App version of Cluedo optimised for holiday entertainment. Written in vanilla JavaScript for maximum performance and supports 20+ concurrent users. The Fisher-Yates shuffle algorithm ensures fair card distribution and collision detection handles game piece movement. Service workers enable offline play so the game works without internet connection.

What I Built:
  • 20+ concurrent user support
  • Offline gameplay capability
  • Fisher-Yates shuffle algorithm for fair play
